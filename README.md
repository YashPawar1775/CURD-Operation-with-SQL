# CURD-Operation-with-SQL:Node.js, Express.js(ejs), MySQL
All the CURD operations are implemented with the database connectivity of SQL Database. The random data is generated with the help of Faker and displayed with help of ejs.

# Download Requirements
Downlaod the extensions mentioned in package.s.json using command "npm i install" to run the projet without errors.

# SQL Connections
Basic SQL connection string format is given in the index.js file comeplete the details with your database to connect the webpage to the database.
